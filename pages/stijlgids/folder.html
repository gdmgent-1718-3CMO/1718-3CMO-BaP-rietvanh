Folder

De folder wordt gebruikt als communicatiemiddel naar de buitenwereld toe. De folders worden geplaatst op plaatsen die relevant zijn voor onze activiteiten.

Een template van de folder kan hieronder gedownload worden. De template is een word-bestand zodat de clubleden de inhoud van de folder gemakkelijk kunnen aanpassen. Een InDesign-bestand kan ook gedownload worden voor diegenen met software kennis.

Specificaties

Enkel de inhoud van de folder mag aangepast worden. De illustraties maken deel uit van de huisstijl en moeten behouden blijven. De contactgegevens blijven ook ongewijzigd.

Specificaties folder:

  • 3-luik
  • Zigzagvouw
  • Afmeting afgewerkt: 297mm (99mm + 99mm + 99mm) x 210mm
  • Bleed: 3mm
  • Recto/Verso quadri CMYK

Specificaties typografie:

  • Alineawit InDesign: 3mm
  • Alineawit Word: afstand na alinea: 6pt
  • Kleurcodes vindt u terug onder 'Kleur' in de navigatieruimte links
